Course Conent : ITMD 541 ( Web App Foundations) is a course in which we are going to learn about different protocols and different langauges such as HTML, CSS, Javascript.

The Course Outcomes for this subject are:

  1. Recognize HTML, CSS, and JavaScript markup and code in a web page/application
  2. Describe the differences between server and client-side technology as it relates to applications delivered through internet browsers
  3. Explain the history, role, and use of enabling technologies in modern web-based applications, including JavaScript, Cascading Style Sheets (CSS), and HTML including their impact
  4. Outline strategies for turning interface mockups into working code prototypes
  5. Explain the role of the HTTP and HTTPS protocol and the request/response cycle
